@xlfont/choose

@xlfont/choose is a javascript library that helps you implement the font picker in your webapp. It contains two parts of code:

  • Font Thumbnail Sprite + Metadata Generator
  • Font Picker JavaScript Library

It uses @xlfont/load to load fonts, which take care of the mechanism of partial loading of large font file made for CJK glyphs.

Usage

Download and include index.css, xfl.js and index.js in your html:

<link rel="stylesheet" type="text/css" href="index.css"/>
<script src="xfl.js"></script>
<script src="index.js"></script>

then, prepare font informations files. This includes a font preview sprites image sprites.min.png and font metadata meta.json - which could be built with @xlfont/choose too - and put them in your website.

Next we initialize the fontchooser:

var chooser = new ChooseFont({
  node: <css-selector-for-your-popup-modal>,
  metaUrl: <path-to-your-meta.json>,
  base: <root-directory-for-your-fonts>
});
chooser.init();

choosefont.css uses sprite.min.png so you also have to patch the css rules to let it know the location you place sprite.min.png. For example, add following CSS rule in your css file:

.choosefont .item .img {
  background-image: url(<path-to-your-sprite-png>/sprite.min.png);
}

choosefont will fire choose event when user picks a font and pass the corresponding font object as parameter. We can load the font within the event handler with xl-fontload:

chooser.on("choose", function(font) {
  font.sync(document.body.innerText, function() {
    document.body.style.fontFamily = font.name;
  });
});

Styling

Choosefont.js builds a list of font preview for you, while you can customize it by yourself. This includes two options when initializing choosefont:

grid

type grid @xlfont/choose will initialize a html layout like following:
<div>
  <div class="line">
    <div class="item">
      <div class="inner">
        <div class="img"></div> <!-- the font preview image -->
        <span> Some Font Name </span> <!-- name of this font -->
      </div>
    </div>
    ... <!-- default 4 items per line.could be tweaked by option cols. -->
  </div>
</div>

list

type list @xlfont/choose will initialize a html layout like following:
<div>
  <div class="item">
    <div class="inner">
      <div class="img"></div> <!-- the font preview image -->
      <span> Some Font Name </span> <!-- name of this font -->
    </div>
  </div>
</div>

Custom Class Name

You can also add custom class name in the item div via itemClass option.


Custom List

You can customize font list by providing two additional config functions:

  • disableFilter(f, idx) - if return true for certain f/idx, the corresponding font will be disabled ( shown gray )
  • defaultFilter(f, idx) - if return false for certain f/idx, the corresponding font won't be shown in the list.

the f object contains following member attributes:

  • name - font name, string.
  • family - font family, array of string.
  • category - font category, array of string.
